emoMint

Facial Recognition App

When a user uploads a picture, the facial recognition software would register if they were happy, content, angry, or sad and then pull from Spotify and Omdb's api to suggest 3 songs and 3 movies that could help them change their mood.

We used the mySQL database to store the daily/momentary uploads and journal entries that would allow the user to look back on any given day/date to see how they were feeling and what was recommended to change their mood.
